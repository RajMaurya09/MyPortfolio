# **App Name**: Raj's Crystal Portfolio

## Core Features:

- Home Page: Showcase a hero section with a typewriter-style introduction of Raj Maurya.
- About Page: Detail Raj Maurya's background, skills, and experience with a clean and informative layout.
- Projects Page: Display Raj Maurya's projects with transparent, glowing cards, descriptions, and links to live demos/repositories.
- Contact Form: Enable users to send Raj Maurya messages via a contact form that utilizes the Node.js backend email API.
- Light/Dark Mode Toggle: Allow users to switch between light and dark themes for optimal viewing experience.
- Smooth Animations: Implement Framer Motion for smooth page transitions and interactive animations.

## Style Guidelines:

- Primary color: Light blue (#89CFF0), providing a modern, airy, transparent aesthetic.
- Background color: Very light grayish-blue (#F0F8FF), maintaining a soft, subtle contrast.
- Accent color: Pale cyan (#A0D6B4), complementing the primary hue, suggesting technical skills and creativity, and creating depth within the transparent elements.
- Body font: 'Inter', a sans-serif font known for its readability and clean design.
- Headline font: 'Space Grotesk', a bold sans-serif font that exudes a tech-oriented style; for the purpose of headlines
- Use clean, minimalist icons that complement the glassmorphism style.
- Implement a responsive grid layout to ensure the portfolio looks great on all devices.
- Use subtle animations to create a smooth and engaging user experience. Page transitions should be fluid and not jarring.
